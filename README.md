# TechMaster Academy — Coaching Center Website

A modern, premium coaching center website for **DevOps, Cloud Computing, Linux & System Administration** courses.

## 🚀 Live Demo
> Deployed via GitHub Pages

## 📁 Project Structure
```
├── index.html      # Main HTML
├── style.css       # Styles (dark theme, animations)
├── script.js       # Interactivity (terminal, particles, slider)
└── README.md
```

## ✨ Features
- 🌑 Stunning dark theme with glassmorphism
- ⌨️ Live animated terminal in the hero
- 🎯 Floating particle animation
- 📊 Animated stat counters
- 🃏 Course cards with glow effects
- 💬 Auto-rotating testimonial slider
- 📜 Scroll-reveal animations
- 📱 Fully responsive (mobile-first)
- 📬 Contact/Enquiry form

## 🛠️ Deploy to GitHub Pages

1. Create a new GitHub repository
2. Push all files to the `main` branch
3. Go to **Settings → Pages**
4. Set source to **Deploy from a branch → main → / (root)**
5. Your site will be live at `https://<your-username>.github.io/<repo-name>/`

## 📞 Customization
Edit the following in `index.html`:
- Phone number, email, location
- Social media links
- Instructor name / academy name
- Course fees / batch timings
